Compress Image

Overview


Optimized image processing for web apps, converting uploads to WebP with customizable compression


API Details

Endpoint: /upload/
Method: POST
Input: Image file (PNG, JPEG)
Output: Compressed WebP image


Features


  • Ensures the uploaded file is an image. Returns a 400 error if it's not.
  • Converts the image to WebP format with minimal quality loss and efficient compression.

Required Libraries


  • Pillow For image conversion
  • FastAPI For handling the API requests
